# Installation
## Ubntu18.04
```bash
$ cd ~
$ git clone --recursive https://github.com/GodRyosuke/WanderTypingGame.git
$ cd WanderTypingGame
$ mkdir build && cd build
$ cmake ..
$ make
$ cd src/test
$ ./opengl_test
```
